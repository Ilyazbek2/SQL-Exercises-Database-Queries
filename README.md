# SQL Exercises: Database Queries

This repository contains a collection of SQL exercises solved step by step. The exercises cover:

- Aggregation functions (COUNT, AVG, SUM)
- Joins and table relations
- Filtering and date operations
- Percentages and rounding
- Classroom and school database examples
- Purchasing and payments examples

## How to Use

1. Load the sample data from the `data/` folder into your database.
2. Open the SQL scripts in the `sql/` folder.
3. Execute queries to see results.
4. Read explanations in `docs/explanations.md`.
